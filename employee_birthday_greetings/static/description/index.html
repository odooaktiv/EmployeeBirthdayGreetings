Employee Birthday Greetings

This module will help to sent an automated Birthday Greetings to the employees if there Birth Date is set in the Contact Form.

Prerequisites before Installation

No prerequisites required before installation.

Configurations

You need to predefine the Birthday mail format by going to The Company Settings and adding the Email Format which should be used in the Birthday Greetings Mail or in User Setting after installing the module and set the Template for the Birthday mail under the Birthday Email Format tab.

Step-By-Step Guidelines

When an employee Birthday is set in the contact form on his birthday an auto generate mail is passed on the the employee with the customized Birthday wishes.

You can set the Birthday Email Format differently for each users.By going to Settings -> Users. For Example if the Ashley Presley birthday is there than the email will be sent with the format which is set in her User form.

If there is no Email Format set for the User whose birthday is there the default format will be taken from the company's form.
