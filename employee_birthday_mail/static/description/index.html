Employee Birthday Mail

This module will help to sent an automated Birthday Mail to the Birthday employee if it's Birth Date is set in the Contact Form.

Prerequisites before Installation

No prerequisites required before installation.

Configurations

You need to predefine the Birthday mail format by going to User Setting after installing the module and set the Template for the Birthday mail under the Birthday Email Format tab.

Step-By-Step Guidelines

When an employee Birthday is set in the contact form on his birthday an auto generate mail is passed on the the employee with the customized Birthday wishes.
